# SpaceShooter

Simple spaceshooter game.

## Compatible inputs

- Keyboard
- Mouse and touch

## Compile and run

### Requirements

In order to compile the project, it is recommended:

- Java 8+ (Recommended 11)
- Gradle 4.x or 5.x for Java 9+
- IDE with Gradle support

### Run

Tested with Netbeans 11 (incubating) - Desktop project.
